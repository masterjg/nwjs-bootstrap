# NW.js bootstrap

NW.js bootstrap to kickstart your project.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

```
Node ^11.10
NPM ^6.7.0
pnpm ^2.25.6
```

### Installing

Install pnpm

```
npm install -g pnpm
```

Install everything else

```
pnpm install
```
